# DBExperiment_Electron
数据库原理课设 with electron@NUAA 2018

## Install
```sh
# mysql client
# user:root
# password:root123456
$ git clone https://github.com/DoubleMice/DBExperiment_Electron.git
$ cd DBExperiment_Electron
$ mysql -u root -p
$ password: root123456
$ mysql source dbExp.mysql
$ npm install
$ npm start
```
